Leecode 回溯 组合总和

发表于2023-04-07更新于2023-04-07

引用自代码随想录

描述

给定一个无重复元素的数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的数字可以无限制重复被选取。

说明：

  • 所有数字（包括 target）都是正整数。
  • 解集不能包含重复的组合。

示例 1：

  • 输入：candidates = [2,3,6,7], target = 7,
  • 所求解集为： [ [7], [2,2,3] ]

示例 2：

  • 输入：candidates = [2,3,5], target = 8,
  • 所求解集为： [ [2,2,2,2], [2,3,3], [3,5] ]

提示：1 <= candidates[i] <= 200

注意图中叶子节点的返回条件，因为本题没有组合数量要求，仅仅是总和的限制，所以递归没有层数的限制，只要选取的元素总和超过target，就返回！本题还需要startIndex来控制for循环的起始位置，为什么呢？注意题目中的要求，解集不能包含重复的组合，以2，5，3为例，如果选择5后，还能选择2，那么就和先选择了2，然后自然选5重复了，为了避免重复，所以我们要设置一个startIndex，同样的因为可以无限选择同一个元素，所以startIndex需要定在已经选择过的上个元素，如2，5，3，选择5后，先一层的startIndex还是从5开始，这样就可以无限选择5了。
